# How to install Google Chrome in Kali Linux

> Google Chrome is a cross-platform and free web browser that is widely used by regular users and tech enthusiasts alike. In this tutorial, you will learn how to install Google Chrome on Kali Linux.

## Step 1: Update Kali Linux

To start off, we need to update the system packages and repositories. Itâ€™s always a good idea before getting started with anything else and so with that, launch your terminal and run the command:

```bash
sudo apt update
```

## Step 2: Download Google Chrome Package

Click the [Link](https://www.google.com/chrome/) to visit the official site.

## Step 3: Click on download chrome.

a pop up will show like this
![image](https://user-images.githubusercontent.com/85288261/204074675-8f2fdf2f-4672-4c89-92d0-97a787a87f2b.png)
 
Select .dev and accept and then install

## Step 4: Installation

Open your file manager and nevigate to download section

![image](https://user-images.githubusercontent.com/85288261/204074748-7eb21edc-fbca-4f81-b556-eddfe1b8fa21.png)

And right click and select **open in terminal**

After that type the commands:

```bash
ls
```
It will show you all the files that are in that directory

then type:

```bash
sudo apt install ./<filename>.deb
```

And wait for some little amount of time, then you will see your chrome browser is downloaded
